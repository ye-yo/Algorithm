## 정렬

#include<algorithm>

```cpp
sort(배열의 시작주소, 배열의 마지막 주소);

//배열일 경우 배열 변수는 포인터이므로
sort(배열명, 배열명+ 배열 크기);
sort(arr, arr+10)

//vector일 경우
sort(v.begin(), b.end())
```

정렬 범위는 start <= arr <end 이기 때문에 배열의 크기를 넣어줘야 함.

sort()함수는 기본 오름차순 정렬

### 내림차순 정렬

sort(start,end,grearter<자료형>())

### 임의의 조건 기준으로 정렬

sort(start,end,compare)를 이용하면 사용자가 정한 함수(compare)를 기준으로 정렬

### 사전 순 정렬

```
a < b
```
